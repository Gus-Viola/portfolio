What Have I Built?

A highlight of my projects.

Tracker Angular App

Github repo for this project Live Demo

Tech stack: Angular, NPM, JavaScript, Node.js; HTML5, CSS

AWS back-end: Cognito, Lambda, DynamoDB, API Gateway, Cloudfront, Route53, S3

Tracker is a single-page Angular app which allows a logged-in user (via AWS Cognito) to privately save some personal information and compare to that of other users. The app uses an API Gateway I configured in AWS to access a Lambda function, which stores the information in a NoSQL DynamoDB database.

Tracker is a 2021 project designed by Maximilian Schwarzm√ºller in his course AWS Serverless APIs & Apps. The Angular front-end code is his and I adapted it, and connected to the back-end coded by me.

This project showcases how I can setup and code with AWS Cognito, Lambda, DynamoDB and the API Gateway for a CRUD application.

MyReads React App

Github repo for this project Live Demo

Tech stack: React, NPM, JavaScript, API; HTML5, CSS

AWS back-end: S3, Cloudfront, Route53

MyReads is a single-page React app which allows you to search books from a Udacity database. The books can be stored in three different shelves, and moved around. It is a showcase for my skills using the React DOM.

MyReads is a 2018 project for my Udacity Nanodegree in Web Front-end, and was built upon the starter template found here. I added interactivity to the app by refactoring the static code from the template.
